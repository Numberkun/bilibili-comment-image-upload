# B站评论区增强图片上传

为 B站（bilibili）评论区图片上传按钮增加**拖拽、粘贴、多选**功能，替代浏览器默认的单文件选择对话框。

## 功能

- **点击上传按钮弹出自定义窗口**，替代系统原生文件选择器
- **拖拽上传** — 直接将图片文件拖入窗口即可添加
- **粘贴上传** — 在弹窗内（或弹窗打开时的页面任意位置）按 `Ctrl+V` 粘贴剪贴板中的图片
- **多图选择** — 文件选择器支持一次选中多张图片
- **缩略图预览** — 横向卡片式排列，文件名自动截断，可单独移除
- **批量上传** — 点击"上传"后逐张注入 B站上传管线，显示进度"上传中 2/5"
- **上限感知** — 打开弹窗时自动检测评论区已上传图片数，弹窗内最多补齐到 9 张

## 安装

1. 浏览器安装 [Tampermonkey](https://www.tampermonkey.net/) 或 Violentmonkey 扩展
2. [点击安装此脚本]() <!-- 替换为实际安装链接 -->
3. 访问任意 B站视频页面，在评论区输入框展开工具栏，点击图片按钮即可

## 使用说明

| 操作 | 说明 |
|------|------|
| 点击图片按钮 | 弹出上传窗口 |
| 点击「选择图片」 | 打开文件选择器（支持多选） |
| 拖入图片 | 将图片文件拖入虚线区域 |
| Ctrl+V | 弹窗打开时粘贴剪贴板中的图片 |
| 点击 × | 移除单张图片 |
| 点击「+ 添加更多」 | 追加更多图片（不超过剩余配额） |
| 双击图片区域 | 快捷上传 |
| 点击「上传」 | 逐张注入 B站上传管线 |
| Esc / 点击遮罩 | 关闭窗口 |

## 兼容范围

- `https://www.bilibili.com/video/*`
- `https://www.bilibili.com/bangumi/play/*`
- `https://www.bilibili.com/list/*`

支持 B站评论区懒加载、SPA 页面切换、主评论框和回复框。

## 注意事项

- 实际上传仍走 B站原生接口，图片格式和大小限制与 B站一致（JPG / PNG / WEBP / BMP / GIF，单张 ≤ 20MB）
- 评论区最多上传 9 张图片，弹窗已上传张数不会被计入新限制
- 若 B站 UI 改版导致按钮失效，请反馈更新
